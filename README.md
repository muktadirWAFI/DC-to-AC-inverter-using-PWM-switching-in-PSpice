# DC to AC Inverter Using PWM Switching

This repository contains a project on a **DC to AC inverter using PWM (Pulse Width Modulation) switching**. 
The circuit is designed and simulated in **PSPICE** to convert a DC input source into an AC output waveform.
The project uses an **H-bridge inverter configuration** with PWM control to synthesize an alternating output voltage.  

The inverter operation is based on:
- an **H-bridge switching structure**
- a **sinusoidal reference waveform**
- a **high-frequency triangular carrier waveform**
- switching logic produced by comparing the reference and carrier signals

## Objectives

The main objectives of this project are:
- to understand the principle of DC to AC inversion
- to understand PWM switching
- to design a PWM-based inverter circuit
- to simulate the inverter in PSPICE
- to analyze the output characteristics of the inverter

## Working Principle

The report describes the working principle in the following stages:

### 1. H-Bridge Configuration
The inverter uses an H-bridge with four switches. These switches are arranged to allow bidirectional current flow through the
 load and generate an alternating output.
### 2. PWM Generation
PWM is generated by comparing:
- a **sinusoidal reference signal**, which defines the desired output waveform
- a **triangular carrier signal**, whose frequency is much higher than the output frequency 
When the carrier waveform is greater than the reference waveform, one switching state is selected; otherwise,
 the opposite state is selected. This comparison creates the PWM switching pulses.

### 3. Switching Operation
The switches operate in complementary pairs. The report notes that switch 1 and switch 2 work in reverse states, 
and switch 3 and switch 4 also work in reverse states. This complementary switching produces the alternating waveform across the load.

### 4. Output Voltage Synthesis
The PWM-controlled switching produces a modified AC waveform across the load. T
he duty cycle determines the effective output voltage. After low-pass filtering, the waveform becomes closer to
 a sinusoidal output

### 5. Filtering
Because PWM introduces harmonic content, a low-pass filter is used to reduce higher-frequency
 components and obtain a smoother sinusoidal AC waveform. The report includes filter calculations for both 50 Hz and 60 Hz operation.
